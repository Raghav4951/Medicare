MediCare is a web-based healthcare management system built using PHP and MySQL. It provides an admin dashboard to manage patients, doctors, appointments, and doctor join requests efficiently.

## 🚀 Features

- 🔐 Admin Authentication (Login System)
- 👨‍⚕️ Doctor Management
- 📝 Doctor Join Request Approval/Rejection
- 🧑 Patient Management (View, Delete, Status)
- 📅 Appointment Tracking
- 📊 Dashboard Overview with Stats
- ✅ Status Management (Active / Blocked)
- 📱 Responsive UI (Works on different screen sizes)

## 🛠️ Tech Stack

- PHP (Core PHP)
- MySQL (Database)
- HTML, CSS (UI)
- JavaScript (basic interactivity)

## 📂 Modules

- Admin Dashboard
- Patients Management
- Doctors Management
- Doctor Requests
- Appointments

## 🔒 Security

- Prepared Statements (SQL Injection Protection)
- Session-based Authentication
- Input Validation

## 📌 Future Improvements

- AJAX-based actions (no page reload)
- Role-based access (Admin/Doctor/Patient)
- API integration
- Better mobile responsiveness

## ⚙️ Setup Instructions

1. Clone the repository
2. Import the `.sql` file into phpMyAdmin
3. Configure `db.php` with your database credentials
4. Run the project on XAMPP / local server
